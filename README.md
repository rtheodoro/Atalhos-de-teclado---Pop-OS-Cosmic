# Atalhos de teclado - Pop_OS - Cosmic

**Keyboard shortcuts** - Armazena meus atalhos de teclado customizados para serem importados quando precisar formatar o sistema.

#### Exportando os atalhos customizados

- Acesse: `/home/your_user/.config/cosmic/com.system76.CosmicSettings.Shortcuts/v1/custom`
- Copie o arquivo `custom`
- Salve em algum lugar

#### Importando os atalhos customizados

- Acesse: `/home/your_user/.config/cosmic/com.system76.CosmicSettings.Shortcuts/v1/custom`
- \*Cole o arquivo `custom`

\* Antes de importar, verifique se o brilho de tela é alterado nativamente com `Fn+F1` ou `Fn+F2`. Caso funcione, não esqueça de alterar esta customização.

#### Lista de atalhos modificados

| Ação                                            | Atalhos                                                           |
| ----------------------------------------------- | ----------------------------------------------------------------- |
| Mover janela para baixo                         | Super+Shift+z ou Super+Shift+Down                                 |
| Mover janela para a esquerda                    | Super+Shift+h ou Super+Shift+s                                    |
| Mover janela para a direita                     | Super+Shift+Right ou Super+Shift+d                                |
| Mover janela para cima                          | Super+Shift+Up ou Super+Shift+a                                   |
| Mover a janela para a área de trabalho anterior | Super+Ctrl+Shift+Left ou Super+Ctrl+Shift+h ou Super+Ctrl+Shift+k |
| Mover a janela para a próxima área de trabalho  | Super+Ctrl+z ou Super+Ctrl+Shift+Down ou Super+Ctrl+Shift+l       |
| Mover a janela um monitor à esquerda            | Super+Alt+Shift+s ou Super+Ctrl+l+s                               |
| Mover a janela um monitor à direita             | Super+Alt+Shift+l ou Super+Ctrl+s+d                               |
| Mover a janela um monitor acima                 | Super+Alt+Shift+Up ou Super+Alt+Shift+k                           |
| Focar a área de trabalho anterior               | Super+Ctrl+Up ou Super+Ctrl+Left ou Super+a                       |
| Focar a próxima área de trabalho                | Super+z ou Super+Ctrl+j ou Super+Ctrl+Right                       |
| Alternar para a saída acima                     | Super+Alt+Up                                                      |
| Alternar para a saída abaixo                    | Super+Alt+Down                                                    |
| *Reduzir o brilho da tela                       | Desativado                                                        |
| *Aumentar o brilho da tela                      | Desativado                                                        |
| Silenciar entrada do microfone                  | Super+F1                                                          |


\*Ver [Como controlar o Brilho de Tela](https://plus.diolinux.com.br/t/como-controlar-o-brilho-de-tela-no-pop-24-04-cosmic/79713/22)
